## 键盘快捷键

项目支持以下键盘快捷键，提升用户体验：

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `ESC键` | 返回顶部 | 快速回到页面顶部 |
| `空格键` | 一键定位当前版本 | 快速滚动到当前正在进行的版本位置 |
| `Ctrl + F` | 聚焦搜索框 | 快速开始搜索版本 |

### 空格键一键定位功能

当页面中有多个版本信息时，按下**空格键**可以：
- 自动平滑滚动到当前版本位置
- 当前版本卡片会有高亮动画效果
- 显示操作反馈提示
- 方便用户快速找到当前版本信息

**使用场景**：
- 页面滚动后需要快速回到当前版本
- 在移动设备上浏览时快速定位
- 对比不同版本信息时快速切换
